### Criar uma conta
Para adicionar o Utmify em sua loja, é necessário criar uma conta.

1. Acesse o site oficial do [Utmify](https://utmify.com.br).
2. Procure a opção de cadastro e preencha com os dados necessários.

### Configurando no painel
1. Acesse o painel da CentralCart: [Configurações -> Widgets](https://app.centralcart.com.br/settings/widget)
2. Adicione o **API Token**.

<Frame>
  ![Email](/images/shared/utmify2.png)
</Frame>

### Como conseguir o API Token

1. Acessa o painel do [utimify](https://app.utmify.com.br/).
2. Navegue até: **Integrações** -> **Webhooks** -> **Adicionar Credencial**, depois adicione um nome e selecione: **Criar credencial**.

<Frame>
  ![Email](/images/shared/utimify.png)
</Frame>

3. Pronto, basta copiar o **API Token** e adicionar no painel.

  ![Email](/images/shared/utmify3.png)
</Frame>
