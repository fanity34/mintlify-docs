## Baixar script

1. Faça login no seu [painel de controle](https://app.centralcart.com.br).
2. Navegue até [Configurações -> Integrações -> Conectar ao RedM](https://app.centralcart.com.br/settings/plugin/).
3. Agora, basta clicar em **download** que o script será baixado diretamente para o seu computador.

<Frame>
  ![Download](/images/fivem/download-script.png)
</Frame>

## Instalação no servidor

1. Para instalar nosso script em seu servidor, basta colocá-lo na pasta de resources.

<Frame>
  ![Resources](/images/redm/script.png)
</Frame>

2. Após isso, abra a pasta, encontre e abra o arquivo `config.js` e insira o token (chave privada) obtido no painel.

<Frame>
  ![Config](/images/fivem/config-fivem.png)
</Frame>

3. Em seguida basta colocar para iniciar o script em seu servidor. Para adicionar basta inserir a linha abaixo no arquivo `server.cfg` ou `resources.cfg`:

```javascript server.cfg
ensure centralcart
```
