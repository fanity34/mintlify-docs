- `token` **->** o token é usado para o script estabelecer a conexão com a sua loja
- `framework`**->** defina se o script irá detectar automaticamente ou usar uma framework específica. veja em [opções](#opcoes)
- `commands`
  - `vip` **->** defina qual comando será utilizado para ver os benefícios temporários do jogador
- `game`
  - `vehicle_table` **->**  defina qual tabela de veículos será utilizada
  - `homes_table` **->**  defina qual tabela de casas será utilizada
- `chat`
  - `enabled` **->** se ativo, o script irá enviar uma mensagem global no chat
  - `message` **->** a mensagem que será enviada para todos os jogadores
  - `style` **->** o código css para personalização da mensagem global
- `notification` **->** configure quais serão os comandos usados in-game
  - `enabled` **->** se ativo, o script irá exibir uma notificação na tela do comprador
  - `type` **->** o tipo da notificação. veja as [opções](#notificacoes)
  - `content`
    - `title` **->** a mensagem que será exibida para o comprador
    - `position` **->** a posição na tela que irá aparecer. veja as [opções](#opcoes)
- `debug` **->** opção para ver logs de depuração

## Notificações
O script tem duas notificações in-game, Default e Party, ambos você pode ativar na configuração do script.

<Tabs>
  <Tab title="Default">
    Ative a notificação **Default** substituindo a configuração por:

    ```js    
    notification: {
      enabled: true,
      type: "default", 
      content: {
        title: "A compra de {packages} foi concluída!",
        position: "top-right", 
      },
    },
    ```

    <Frame>
      ![Notificação in-game](/images/fivem/noti.png)
    </Frame>
  </Tab>

  <Tab title="Party">
    Ative a notificação **Party** substituindo a configuração por:
    ```js
    notification: {
      enabled: true,
      type: "party",
      content: {
        title: "A compra de {packages} foi concluída!",
        subtitle: "Obrigado por ajudar a cidade!",
      },
    },
    ```

    <Frame>
      ![Notificação in-game](/images/fivem/noti2.png)
    </Frame>
  </Tab>
</Tabs>
