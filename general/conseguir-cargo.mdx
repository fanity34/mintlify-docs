## Conseguindo o ID de cada cargo
Para obter o ID de cada cargo, você precisará primeiro ativar o Modo Desenvolvedor em sua conta do Discord. Siga as etapas abaixo:

1. Acesse a **configurações da sua conta** no Discord.
2. Clique na seção **avançado**.
3. Habilite o **modo desenvolvedor**.

<Frame>
![Modo desenvolvedor](/images/shared/cargos-discord.png)
</Frame>

Após ativar o Modo Desenvolvedor, siga as instruções a seguir para obter o ID de cada cargo:

1. Acesse o seu servidor no Discord.
2. Abra o menu de **configurações**.
3. Selecione a opção **cargos**.
4. Escolha o cargo para o qual deseja obter o ID.
5. Clique com o botão direito do mouse sobre o cargo selecionado.
6. Escolha a opção de copiar o ID.

</Frame>
