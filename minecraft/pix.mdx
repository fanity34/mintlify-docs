## Passo a passo (Versão escrita)
O sistema tem o intuito de facilitar a compra de beneficios in-game, otimizando a experiência do jogador e impulsionando a taxa de conversão.

<Frame>
  ![QRCODE do pix](/images/minecraft/pix.png)
</Frame>

## Baixar o plugin

1. Faça login no seu [painel de controle](https://app.centralcart.com.br).
2. Navegue até [Configurações -> Integrações -> Conectar ao Minecraft](https://app.centralcart.com.br/settings/plugin/minecraft).
3. Agora, basta clicar em **outros downloads** e seleciona **CentralCart-Pix** que o plugin será baixado diretamente para o seu computador.

<Frame>
  ![Baixar o plugin no seu servidor](/images/minecraft/pix-minecraft.png)
</Frame>

<Note>
O plugin para integração com pix in-game só irá funcionar, se o plugin principal da Central estiver instalado em seu servidor.
</Note>

## Configuração do plugin

1. Adicione o plugin da CentralCart em seu servidor.
2. Reinicie ou ligue o servidor para o plugin carregar.
3. Após o plugin ser iniciado, uma pasta de configuração será gerado.

<Frame>
  ![QRCODE do pix](/images/minecraft/plugins.png)
</Frame>

4. Acesse a pasta **CentralCart-Pix** e edite o arquivo **config.yml**.

## Exemplo de configuração

<Frame>
  ![menu](/images/minecraft/menu-pix.png)
</Frame>

```yml
Categories:
  vips: # Nome da categoria
    Slot: 1 # Slots onde a categoria vai ficar no menu
    ID: 388 # ID do item
    Data: 0
    Name: '&aVIPs' #Titulo da categoria
    Lore: # Descrição da categoria
      - ''
      - '&eClique para selecionar.'
    Products:
      vip-carvao: # Nome do produto
        Slot: 28 # Slots onde o produto vai ficar no menu
        ID: 351 # ID do item
        Data: 10
        Product-ID: 174876 # ID do produto
        Name: '&aVIP' # Titulo do produto
        Lore: # Descrição do produto
          - '&8Pemanente'
          - ''
          - '&a✔ &7Voar no Lobby'
          - '&a✔ &7Criar clans'
          - ''
          - '&7Preço: &a<value>'
          - ''
          - '&eClique para comprar.'
        Advantages:
          - '     &a&lVIP &c(X)'
          - ''
          - '&a✔ &7Voar no Lobby'
          - '&a✔ &7Cosméticos'
          - '&a✔ &7Skin Customizada'
```
