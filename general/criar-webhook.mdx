## Criando webhook
1. Acesse o seu servidor no Discord.
2. Abra as configurações do canal que deseja criar o webhook.
3. Seleciona: **Integrações > Criar webhook**.

<Frame>
![Modo desenvolvedor](/images/shared/webhook.png)
</Frame>

4. Agora basta criar um nome no webhook e copiar a URL.

<Frame>
![Modo desenvolvedor](/images/shared/url-webhook.png)
</Frame>

5. Pronto, basta adicionar a url do webhook no painel.

<Frame>
![Modo desenvolvedor](/images/shared/urlwebhook.png)
</Frame>
