## Como ativar widget do seu servidor.
O widget Discord é um widget plug-and-play que você pode incorporar em um site para exibir seus servidores Discord diretamente no site.

1. Acesse o seu servidor no Discord.
2. Abra o menu de **configurações**.
3. Selecione a opção **Widget**.

<Frame>
![Cargos do Discord](/images/shared/widgets.gif)
</Frame>

4. Pronto, basta adicionar o ID do servidor no painel da CentralCart.
