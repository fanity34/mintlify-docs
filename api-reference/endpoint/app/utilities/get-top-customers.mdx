title: "Top Customers"
api: "GET https://api.centralcart.com.br/v1/app/widget/top_customers"

description: "Retorna a lista dos top compradores."

<ParamField query="from" type="string" required>
  Data de início para a busca. Ex. `2024-12-01`
</ParamField>

<ParamField query="to" type="string" required>
  Data de término para a busca. Ex: `2024-12-30`
</ParamField>

<Info>Este recurso lista os 10 maiores compradores ordenados pelo valor gasto.</Info>
