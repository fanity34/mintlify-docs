## Alteração variável do checkout
A framework QBCore não utiliza o ID padrão, por isso, para funcionar corretamente, será necessário alterar a variável do checkout para Citizen ID.

1. Para isso acesse:  [Configurações -> Variáveis de checkout](https://app.centralcart.com.br/settings).
2. Selecione a variável para realizar a alteração.
3. Altere o tipo para **Texto** e a descrição para **Seu Citizen ID**.

<Frame>
  ![Variável do checkout](/images/fivem/qbcore-checkout.png)
</Frame>

<Warning>
  O tipo **texto** permitirá que o jogador utilize números e letras no checkout de pagamento.
</Warning>

## Comandos
Além dos comandos padrão que vem na configuração do produto, você pode adicionar um salário para o jogador e remove-lo.

O primeiro comando é:
``salary {client_identifier} quantidade``

<Frame>
  ![Comando para adicionar o salário](/images/fivem/comando.png)
</Frame>

<Check>
  O jogador recebe o salário a cada 10 minutos online no servidor.
</Check>

O segundo comando é: ``removeSalary {client_identifier}``

<Frame>
  ![Comando para remover o salário](/images/fivem/comando-remover.png)
</Frame>
