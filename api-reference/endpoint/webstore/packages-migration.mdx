---
title: "Packages Endpoint Migration"
description: "Nota de migração temporária para o endpoint de pacotes."
---

- Durante o período de migração, você pode passar o parâmetro `v=2` na query para obter o **novo formato de resposta**.
Exemplo:
```bash
GET https://api.centralcart.com.br/v1/webstore/package?v=2
```
- Em 15 dias `(15/10/2025)`, o uso de `v=2` não será mais necessário e o novo formato se tornará o **oficial**.
- Veja o exemplo do novo formato de resposta em [List Packages](/api-reference/endpoint/webstore/packages) e adapte seu sistema para suportar o novo modelo.

<Note>Para tirar qualquer dúvida sobre a migração [entre em contato](/suporte).</Note>
