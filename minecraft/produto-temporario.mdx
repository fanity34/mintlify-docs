## Configuração do produto
Para deixar o produto temporário, você precisa configurar um comando para remover o beneficio depois de um período de dias. Iremos mostrar no exemplo abaixo, como você pode fazer a configuração:

### Exemplo

1. Acesse a configuração do seu produto e determine os dias que o beneficio do jogador será removido, na imagem abaixo, definimos como 30 dias.

<Frame>
  ![Produto temporário](/images/shared/produto-temporario1.png)
</Frame>

2. Crie um comando que será executado quando o cliente comprar o produto na loja, no nosso caso iremos configurar para adicionar o cargo ouro após a compra.

<Frame>
  ![Produto temporário](/images/minecraft/comando-produto.png)
</Frame>

<Check>No **Minecraft** cada servidor utilizam comandos diferentes, caso o seu plugin remova o vip de forma automática, não precisa seguir o **próximo passo**.</Check>

3. Agora para o grupo ouro ser removido depois dos **30 dias**, você precisa criar um comando para remover o beneficio com o status **quando expirar**, igual na imagem abaixo:

<Frame>
  ![Produto temporário](/images/minecraft/comando-remover.png)
</Frame>

<Note>Você pode personalizar tempos diferentes para cada comando ao selecionar a engrenagem e especificar os dias que o comando será executado. </Note>
4. Pronto, o grupo **Ouro** será removido após os **30 dias**.

## Conclusão
Comandos com a opção **quando expirar** serão executado conforme a configuração da engrenagem ou por padrão na opção **remover do cliente após**.
