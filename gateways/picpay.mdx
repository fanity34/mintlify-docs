## Criar conta no PicPay Negócios
Para começar a aceitar o PicPay em sua loja, é necessário criar uma conta.

1. Acesse o site oficial do [PicPay Négocios](https://cadastro-negocios.picpay.com/dados-cadastrais).
2. Procure a opção de cadastro e preencha com os dados necessários.

<Warning>
Para abrir uma conta no PicPay Negócios, é obrigatório ter um CNPJ.
</Warning>

## Obter o token de comunicação
O token de comunicação é utilizado para conectar a api do PicPay com a Centralcart. 

1. Navegue até [Painel do Picpay -> Integrações -> Gerar Tokens](https://painel-negocios.picpay.com/integracoes).
2. Pronto! Agora basta copiar e [preencher](https://app.centralcart.com.br/settings/gateway/picpay) o token no painel da CentralCart.

<Frame>
  ![Formulário de aplicação do PicPay](/images/shared/picpay-token.png)
</Frame>

## Vinculando o seu PicPay com a CentralCart
Agora, vamos prosseguir para preencher as credenciais do PicPay.

1. Comece acessando a [configuração do PicPay](https://app.centralcart.com.br/settings/gateway/picpay) na nossa plataforma.
3. Preencha com o [token](#obter-o-token-de-comunicacao) do PicPay e clique em salvar.
