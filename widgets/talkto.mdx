### Criar uma conta
Para adicionar o Tawk.to em sua loja, é necessário criar uma conta.

1. Acesse o site oficial do [Tawk.to](https://www.tawk.to).
2. Procure a opção de cadastro e preencha com os dados necessários.

### Configurando no painel
1. Acesse o painel da CentralCart: [Configurações -> Widgets](https://app.centralcart.com.br/settings/widget)
2. Selecione: **Configurar**

<Frame>
  ![Email](/images/shared/talkto-painel.png)
</Frame>

3. Vai aparecer 2 campos para preencher: **Property ID** e **Widget ID**.

### Como conseguir o Property ID e Widget ID

1. Acessa o painel do [Tawk.to](dashboard.tawk.to/).
2. Navegue até: **Configurações** -> **Overview** e copie o **Property ID**.

<Frame>
  ![Email](/images/shared/talkto-painel2.png)
</Frame>

3. Navegue até: **Configurações** -> **Channels** -> **Chat Widget** e copie o **Widget ID**.

<Frame>
  ![Email](/images/shared/talkto-3.png)
</Frame>
